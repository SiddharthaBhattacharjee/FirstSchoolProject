
~USER MANUEL~

1) To start the app, run the Employee_DB.py in Project_employees file
2) Log in to the app or sign up and continue to menu page
3) User may perform any fungtion given in the menu
4) All the menu options are there to insert,update,display the database of employee
5) Employees table can be updated from display table option directly.

1) MySQL (keep the mysql password handy while running for the first time)
